# Création du jeu Fox and Hound Game en Python
j'ai développer ce jeu en python suivant les règles : https://boardgamegeek.com/boardgame/148180/fox-and-hounds
Ce jeu se joue à 2 joueur.
Il y a plusieurs Hound sur le plateau et un seul Wolf. Les Hounds ne peuvent se déplacer diagonalement
vers le "bas", tandis que le Wolf peut se déplacer digonalement dans tous les sens.
Un Wolf ne peut pas sauter par dessus un Hound tout comme il ne peut pas aller dans une case déjà
occuper. De même pour les Hounds.
Le but du Wolf est d'atteindre l'autre côté du plateau tandis que les Hounds doivent stopper le Wolf 
(le wolf ne doit pas avoir de case possible pendant son tour).
Le premier qui réussi à attendre cet objectif gagne la partie.

### Ce que j'ai appris
Implémenter ces règles en Python à été un bon challenge pour moi. Je me suis familiariser avec les
classes en Python. Réussir à écrire la logique du jeu en code n'était pas facile mais c'était une 
bonne expérience.
